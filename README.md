# 🖥️ COMPUTER

**Commanding Operations Making Progress, Understanding Technological Endeavors, Reporting.**

COMPUTER is a high-end localhost dashboard and system command center designed to provide a unified view of your system health and project development progress. It implements four core pillars of operational management in a futuristic, Iron Man-inspired HUD.

## 📡 Core Pillars

- **🖥️ Commanding Operations**: Real-time system monitoring (CPU, RAM, Uptime) and high-level OS kernels diagnostics.
- **📈 Making Progress**: Automated tracking of workspace milestones. COMPUTER parses your local `TODO.md` to provide a visual completion percentage.
- **🛠️ Understanding Technological Endeavors**: Deep analysis of your workspace, including file count, total storage usage, and programming language distribution.
- **📑 Reporting**: One-click generation of comprehensive status reports for executive summaries or project archiving.

## 🛠️ Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/chambtai-sys/COMPUTER.git
   cd COMPUTER
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Launch the Core**:
   ```bash
   python app.py
   ```

4. **Access the HUD**:
   Open `http://localhost:5000` in your browser.
